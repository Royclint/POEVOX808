## 📘 Description
The VOX808 application is a complete Java messaging system developed for the PROG5121 POE.  
It integrates all three parts of the project:
- **Part 1:** Registration and login validation  
- **Part 2:** Message handling and JUnit testing  
- **Part 3:** Reporting, arrays, and automated CI/CD testing  

This project demonstrates secure user authentication, message management, data storage, and automated testing through GitHub Actions.

---

## ⚙️ Features
- **User Registration:** Name, Surname, Username, Password, Phone Number  
- **Login Validation:** Underscore requirement + password complexity check  
- **Messaging System:** Send multiple messages, view sent messages  
- **File Handling:** JSON storage for messages and user data  
- **Reporting:** Array‑based summary of messages and users  
- **JUnit Testing:** Automated tests for Login, Message, and Report classes  
- **CI/CD Integration:** GitHub Actions workflow (`TestJava.yml`) runs tests on every push
